## NarrativePdfRenderer Implementation

- `render()` calls template.buildSections(), iterates sections calling render(), assembles into pw.MultiPage
- `_buildPageHeader()` returns SizedBox(height: 0) on page 1, template title + accent underline on pages 2+
- `_buildPageFooter()` renders on every page: divider, revision label (left), "Generated by InspectoBot" (center), "Page X of Y" (right)
- Page breaks handled by pw.MultiPage automatic overflow + explicit PageBreakSection (pw.NewPage())
